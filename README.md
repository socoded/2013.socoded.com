# socoded.com

The homepage!

## Setup

```sh
$ bundle install
```

## Deploy

```sh
$ bundle exec rake deploy
```

## Development mode

```sh
$ bundle exec middleman server
```

Then open the page locally at: [http://localhost:4567](http://localhost:4567)
